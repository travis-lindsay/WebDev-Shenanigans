The following is a brief description of the current progresss of the Sudoku game

The following functions/Methods have been fully implemented:

  • pickGame():
    This function takes the game difficulty value picked by the user, and create a game object with the associated values for the difficulty (easy, medium, or hard).
  • giveHint():
    This function finds a randomly generated blank spot in the sudoku grid, place a value in it, and increment the numHints variable.
  • checkValidNumber():
    This function takes the number just inserted into the cell of the table, and check it against the two dimensional array of the correct numbers for that particular puzzle.
  • correctNumber():
    This function is called if the inserted number is in the correct place. It decrements the number of cells left over until the game has been won.
  • wrongNumber():
    This function is called if the inserted number is in the incorrect place. It discards the number, and increase the number of errors in the numCurErrors variable.
  • checkGameWin():
    This function is called from the correctNumber() function, to update and check the numToWin variable to see if the game is now complete.
  • checkGameLose():
    This function is called from the wrongNumber() function, to check the numErrors variable against the max NUMERRORS constant to see if the game is over.
  • gameWin():
    This function is called from the checkGameWin function if the game has been won. It takes the game statistics and resets the game variables, then calls a render function to render the game win overlay.
